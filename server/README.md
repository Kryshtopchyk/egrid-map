## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

For the application to work,
it is necessary to populate the database using paths
/state/populate and /plant/populate

## Routes

Swagger:
All routes are accessible via the path /api
